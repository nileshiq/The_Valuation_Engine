# The_Valuation_Engine
AutoValuate AI predicts car prices using Linear Regression, Random Forest, &amp; Neural Networks. Analyzing 2,000 records, we found Horsepower &amp; Model Year drive value most. Our best model (Linear Regression) achieved 96.6% accuracy, with a low $1,663 average error. A complete data-to-insight pipeline for smart automotive market valuation.
